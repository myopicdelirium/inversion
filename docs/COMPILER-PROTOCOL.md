# The society compiler: parameterization protocol (draft)

How a historical case becomes a configuration, without the fitting ever touching the outcome. Draft 2026-07-24; frozen alongside the first dossier.

## The pipeline

1. **Inputs**: a frozen dossier's parameterization evidence only. The signature (outcome evidence) is sealed: not read, not summarized, not "kept in mind."
2. **Mapping**: every engine axis receives its value or declared distribution from cited evidence via rules written in this protocol, or is explicitly declared unidentified and swept. No axis may be set by feel. The mapping document for a case lists, per parameter: value or distribution, the evidence line it came from, and the rule applied.
3. **The pressure schedule** becomes the world's event timeline (onsets, ramps, snares, famine shadows), inputs never credited as outputs.
4. **Commit before unblinding**: the mapping document, config hashes, seed lists, and the complete runs (or their golden hashes and sweep digests) are committed to git. Only then is the signature opened and the preregistered pass/fail judged, in one pass, with the verdict published either way.
5. **Contrast cases** run under the same protocol with their own mapped parameters; the discrimination requirement (the model must place the case and its contrasts in different regions for independently documented reasons) is judged with the same bars.
6. **Counterfactual lever claims** get their own preregistration before any counterfactual run, and are tested against near-miss records blind where the record allows.

## Statistical form

Every judged quantity declares its seed count, pooling statistic, and replication requirement in the dossier before runs exist. The engine's standing machinery applies: deterministic runs, per-run manifests, byte-stable sweep digests, fresh-seed replication before any claim is packaged, withdrawal on failed replication.

## What this protocol may never do

Tune toward a signature; adjust a mapping rule after unblinding; add axes mid-case; report a pass without the contrast discrimination; describe an unvalidated output as a finding (constitution, standing).
